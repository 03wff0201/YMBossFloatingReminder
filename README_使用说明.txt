圆梦三职业 Boss 悬浮提醒 - 安卓项目说明

一、项目用途
这是一个安卓悬浮窗提醒工具项目。
它只做 Boss 刷新时间提醒和手动记录，不会自动点击游戏，不会自动跑图，不会自动打怪。

二、主要功能
1. 启动悬浮窗
2. 在游戏上层显示一个 Boss 小悬浮窗
3. 自动推荐当前最应该去看的 Boss
4. 手动击杀后，点击“已击杀并记录”
5. 工具自动计算下一轮刷新
6. 支持拖动悬浮窗位置
7. 支持跳过当前目标、切换下一个目标

三、怎么打包 APK
1. 安装 Android Studio
2. 打开本文件夹 YMBossFloatingReminder
3. 等待 Gradle 同步完成
4. 点击 Build > Build Bundle(s) / APK(s) > Build APK(s)
5. 生成 APK 后安装到安卓手机

四、手机上怎么用
1. 安装 APK
2. 打开 App
3. 点击“授权悬浮窗权限”
4. 在系统设置里允许“显示在其他应用上层”
5. 回到 App，点击“启动 Boss 悬浮窗”
6. 切回游戏
7. 点击悬浮窗展开
8. 打完 Boss 后点击“已击杀并记录”

五、注意
不同手机系统可能还需要开启：
- 悬浮窗权限
- 后台弹出界面
- 后台运行
- 锁屏后保持运行
- 电池优化白名单
